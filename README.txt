== Installation ==

Upload the plugin files to the /wp-content/plugins/wp-ai-workflow-automation directory, or install the plugin through the WordPress plugins screen directly.
Activate the plugin through the 'Plugins' screen in WordPress.
Use the Settings -> WP AI Workflow Automation screen to configure the plugin and add your AI API keys.

== Frequently Asked Questions ==
= Do I need coding skills to use AI Workflow Automation? =
No coding skills required! Our visual workflow builder and AI Workflow Generator make it easy for anyone to create complex automations.

= How many AI models can I access in the Lite version? =
The Lite version now includes access to over 300 AI models, including Claude, GPT-4, Gemini, Grok, DeepSeek, and many more!

= Can I create unlimited workflows in the Lite version? =
Yes! The Lite version now supports unlimited workflow creation with no restrictions.

= What is the Workflow Assistant? =
The Workflow Assistant is your 24/7 AI expert that helps you optimize workflows, improve prompts, select the best AI models, and maximize your automation efficiency.

= How does the AI Workflow Generator work? =
Simply describe what you want your workflow to do in plain language, and our AI will automatically create a complete workflow for you. You can then customize it further if needed.

= Can I use templates or share workflows? =
Yes! The Lite version now includes full access to our template library, plus the ability to import and export workflows between sites.

= What AI models are supported? =
We support over 300 AI models including OpenAI (GPT-4, o1, o3), Anthropic Claude, Google Gemini, xAI Grok, DeepSeek, Mistral, Meta Llama, and hundreds more through our integrations.

= Can I use WP AI Workflow Automation on multiple sites? =
Yes! Both Lite and Pro versions can be used across multiple WordPress installations.
